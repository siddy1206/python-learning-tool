# Python Learning Tool

A mini-project using Python and Tkinter to help beginners revise basic Python concepts.

## Features
- Flashcards for core topics such as variables, loops, functions, data types, etc.
- Meanings shown on button click
- Randomized multiple-choice quiz (8 questions)
- Final score screen

## Requirements
- Python 3.x

## How to Run
1. Download the python_learning_tool.py file
2. Run using: `python python_learning_tool.py`
